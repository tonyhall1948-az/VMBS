## 🚀 How to Use

### Option 1: Direct Browser Access
1. Open any `.html` file directly in your web browser
2. All interactive features work without a server
3. Calculators, quizzes, and checklists are fully functional

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if installed)
npx http-server

# Then open: http://localhost:8000
```

### Option 3: Upload to Web Hosting
1. Upload all files to your web host (Hostinger, Bluehost, etc.)
2. Point your domain to the hosting
3. Website is live and accessible globally

### Option 4: Use with Site Builders
- Copy/paste content from HTML files into Wix, Squarespace, WordPress
- Adapt styling to match builder's templates
- Use our content as inspiration

## 📞 Getting Help

### Common Issues

**Q: Calculators not working?**
A: Make sure you're opening the file in a modern browser. Some browsers disable JavaScript for local files. Use a local server instead (see instructions above).

**Q: Styling looks weird?**
A: Clear your browser cache (Ctrl+Shift+Delete) and refresh the page.

**Q: On mobile, navigation is cut off?**
A: This is normal. The mobile nav is responsive. Try scrolling or use the hamburger menu if implemented.
